Email Content Sender - Help

What is this add-in?

This add-in allows you to send the content of the currently displayed email to an API endpoint. It's useful for processing email content through external services.

How to use

  1. Open an email in Outlook
  2. Click the "Send to API" button in the add-in bar
  3. The email content will be sent to the configured API endpoint
  4. Check the status message to confirm the operation was successful

Need more help?

If you're experiencing issues or need additional support, please contact your system administrator.
